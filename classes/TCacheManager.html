implements ICacheManager

TCacheManager.class.php : 28

stores some result in the database along with information which tables and records are connected to the cache.

since 5.0.0 - use chameleon_system_core.cache instead. Do not simply exchange the calls, but instead think about the cache strategy to use. The problem with this implementation was that caching was used too deliberately, leading to way too many cache entries. Instead it is recommended to cache mostly module contents. /

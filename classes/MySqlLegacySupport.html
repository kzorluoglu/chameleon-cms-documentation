MySqlLegacySupport.php : 32

Class MySqlLegacySupport is used to replace the deprecated mysql_ calls by DBAL equivalent calls. Note that many methods are not supported at all.

you should avoid using the database directly - always use it via an abstraction that contains those data access methods you need

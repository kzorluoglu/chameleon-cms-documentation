WebServerExample.php : 105

Check that we have received an IPN notification for the defined event

protected waitForNotificationToBeProcessedBeforeContinuing(string $identifier, string $notificationType) : void

For PHP, there is an IPN handler that will write the contents of the IPN to a file in the format of <amazonOrderReferenceId>_<amazonAuthorizationId>_Authorization. This method will check for the presnece of this file and will loop/timeout until the notification has been handled.

Merchants can use alternative approaches such as memory caches, shared memory or database storage so that scripts serving user pages are able to check on the status of a notification

Parameters
$identifier : string

transaction that we are waiting to query

$notificationType : string

notificationType notification type that we expect

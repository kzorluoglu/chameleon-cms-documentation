IPkgCmsEvent.class.php : 20

defines an event. every event is identified by the context and the event name context: describes what package or subpackage the event belongs to. this helps to isolate events within a package from other events name: with in each context the events are identified by their name.

events may hold data specific to the event. this data can be accessed via GetData

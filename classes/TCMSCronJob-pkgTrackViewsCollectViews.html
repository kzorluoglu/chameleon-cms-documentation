$aTableDateGroups

TCMSCronJob_pkgTrackViewsCollectViews.class.php : 67

this array defines date groups for specific tables.

protected array<string|int, mixed> $aTableDateGroups = array()

you can overwrite it like following for custom usage

$aTableDateGroups = array('cms_document' => array('day', 'week', 'month', 'year', 'all')

'day', 'week', 'month', 'year' and 'all' are valid count periods

'day' => tracks items for every day (date('YmWd')). 'week' => analog (date('YmWxx')) 'month' => analog (date('Ymxxxx')) 'year' => analog (date('Yxxxxxx')) 'all' => total count 'xxxxxxxxxx'
